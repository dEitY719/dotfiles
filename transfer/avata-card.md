# Avatar Card — 입력 초안

> 이 파일은 Avatar Studio UI 입력용 초안이다. 아래 값을 복사해 카드 생성 폼에 붙여넣는다.

## Avatar Card

| 항목       | 값                                                                                                                                                          |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 이름       | AI 기반 풀스택 개발자                                                                                                                                       |
| 책임(목표) | AI coding agent를 협업자로 활용해 풀스택 제품(프론트엔드·백엔드·AI 연계)을 요구사항 분석부터 E2E 검증까지 SSOT·SOLID·TDD 원칙 아래 설계·구현·유지한다. |
| 상태       | active                                                                                                                                                      |
| 역할 목록  | AI 보조 구현자 · 설계·검증 담당 · 개발 워크플로우·환경 관리자                                                                                              |

---

### 역할 연결 순서 (picker 선택 순)

1. `AI 보조 구현자` — 이슈 구현·PR 리뷰 처리 담당
2. `설계·검증 담당` — 요구사항 분석·E2E 검증 담당
3. `개발 워크플로우·환경 관리자` — 스킬 제작·개발 환경 유지 담당
