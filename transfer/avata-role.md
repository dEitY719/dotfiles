# Avatar Role — 입력 초안

> 이 파일은 Avatar Studio UI 입력용 초안이다. Role 3개를 순서대로 생성한 뒤 카드에 연결한다.

---

## Role 1 — AI 보조 구현자

| 항목      | 값                                                                                                                        |
| --------- | ------------------------------------------------------------------------------------------------------------------------- |
| 이름      | AI 보조 구현자                                                                                                            |
| 역할 설명 | GitHub 이슈를 받아 AI coding agent와 함께 설계→구현→커밋→PR까지 수행하고, 자동·수동 리뷰 코멘트를 평가해 반영하는 역할. |
| 업무 목록 | 이슈 분석 및 기능 구현 · PR 리뷰 코멘트 처리                                                                             |

---

## Role 2 — 설계·검증 담당

| 항목      | 값                                                                                                            |
| --------- | ------------------------------------------------------------------------------------------------------------- |
| 이름      | 설계·검증 담당                                                                                                |
| 역할 설명 | 요구사항을 기술 설계로 전환하고, 구현 완료 후 실제 앱에서 E2E 검증을 수행해 PR 머지까지 책임지는 역할.       |
| 업무 목록 | 요구사항 분석 및 기술 설계 · E2E 검증 및 PR 머지                                                             |

---

## Role 3 — 개발 워크플로우·환경 관리자

| 항목      | 값                                                                                                                                |
| --------- | --------------------------------------------------------------------------------------------------------------------------------- |
| 이름      | 개발 워크플로우·환경 관리자                                                                                                       |
| 역할 설명 | superpowers 스킬·dotfiles·Claude Code 플러그인을 제작·개선하고, mise·Docker 기반 개발 환경을 최신 상태로 유지하는 역할.           |
| 업무 목록 | 스킬 작성 및 리팩터링 · 개발 환경 설정 및 마이그레이션                                                                           |
