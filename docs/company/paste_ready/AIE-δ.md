## M-1 조직목표달성 (50%)

### Summary
```text
[K?-δ] 2026 조직목표달성 - OKR 기반 핵심과제 수행
```

### Parent / 팀 Epic 연결
K-1~K-4 중 담당 제품 Story (확정 후 설정). 계층: E4 → SWINNOTEAM-1274 → P-1 → K-? → 본 Story.

### Description
```text
■ 목적
담당 파트원의 2026년 핵심과제를 OKR 프레임워크로 관리하고,
파트 목표 달성에 기여한다.

■ 팀-파트 목표와의 연결
- 팀: E4 (Avatar 7000명) → SWINNOTEAM-1274 (Document Agent, MAU ≥ 5,000명)
- 파트: Story P-1 [AIE O-1] 조직목표달성의 하위 Initiative로 연결
- 개인 OKR이 파트 OKR의 어떤 KR에 기여하는지 명시 필수

■ 평가 기준 (비중 50%)
- 핵심과제 2개 이하로 설정
- OKR 구성: Objective → Key Result → Initiative → KPI
- KPI는 측정 가능한 숫자로 정의
- OKR은 파트장 승인 필요
- 최소 월 1회 1:1 피드백 수행

■ 업무 권한 Level
- L3: 스스로 의사결정을 주도하며 주어진 목표 달성 (결과 부서장 보고)
- L2: 수립한 목표 또는 과제를 리딩하며 동료와 함께 목표 달성
- L1: 부서장/동료의 지시 또는 가이드를 받아 공동의 목표 달성
※ 본 Story 의 각 OKR/Initiative 별로 Level 지정

■ OKR 작성 가이드
- Objective: 달성하고자 하는 목표 (예: AI 서비스 품질 향상)
- Key Result: 목표 달성 측정 지표 (예: 서비스 응답시간 30% 개선)
- Initiative: 구체적 실행 과제 (예: 모델 최적화, 캐싱 도입)
- KPI: 정량 측정 수치 (예: 응답시간 < 500ms)

■ 운영 방식
- 4월: OKR 수립 및 파트장 승인
- 매월: 1:1 feedback (진행률, 장애요인, 조정사항)
- 분기별: KPI 달성률 리뷰

■ 성과 측정
- KPI 달성률 (정량)
- 과제 기여도 (정성)
- 일정 준수

■ 기간
2026.04 ~ 2026.10

※ 하위 Task로 월별 구체적 활동 계획 추가 예정
```
