# using-superpowers
## 개요
superpowers 스킬 체계의 적용 우선순위와 호출 규칙을 정의하는 메타 스킬

## 언제 사용하나
- superpowers 플러그인을 실제 워크플로우에 적용할 때 전반 지침이 필요할 때

## 핵심 절차(번역)
1. 스킬 적용 가능성이 조금이라도 있으면 먼저 확인한다.
2. 사용자 지시 > superpowers 스킬 > 기본 프롬프트 우선순위를 지킨다.
3. 플랫폼별 도구 매핑(Claude/Codex/Copilot/Gemini)을 적용한다.

## 주의/금지 사항(번역)
- 스킬 확인 없이 바로 실행
- 기억에 의존한 구버전 스킬 사용
- 플랫폼 도구 차이를 무시

## 실행 체크리스트
- 작업 시작 전에 이 스킬이 현재 과제에 실제로 필요한지 확인한다.
- 필요하면 해당 스킬의 원문 `SKILL.md`에서 세부 규칙을 다시 확인한다.
- 진행 중에는 체크리스트/게이트 조건을 생략하지 않는다.
- 완료 보고 전에는 관련 검증 명령과 결과를 함께 남긴다.

## 빠른 예시
> "앞으로 작업은 using-superpowers 규칙을 기준으로, 적용 가능한 스킬부터 먼저 판별해줘."
