# /simplify - Code Review and Cleanup

Claude Code 내장(built-in) 스킬. 플러그인/마켓플레이스가 아닌 Claude Code 바이너리에 포함되어 있어 파일시스템에 별도의 `SKILL.md`가 존재하지 않는다.

## 동작 요약

변경된 코드를 3가지 관점에서 병렬 리뷰한 뒤, 발견된 이슈를 직접 수정한다.

## Phase 1: 변경사항 식별

- `git diff` (또는 `git diff HEAD`)로 변경된 코드 확인
- git 변경이 없으면 대화 중 수정된 최근 파일을 대상으로 함

## Phase 2: 3개 리뷰 에이전트 병렬 실행

Agent tool을 사용하여 3개 에이전트를 **단일 메시지에서 동시(concurrent)** 실행한다.
각 에이전트에 전체 diff를 전달하여 완전한 컨텍스트를 제공한다.

| 에이전트                  | 관점      | 주요 체크 항목                                                                             |
| ------------------------- | --------- | ------------------------------------------------------------------------------------------ |
| **Agent 1: Code Reuse**   | 재사용성  | 기존 유틸리티/헬퍼 중복, 인라인 로직이 기존 유틸로 대체 가능한지                           |
| **Agent 2: Code Quality** | 코드 품질 | 중복 상태, 파라미터 스프롤, 복붙 코드, 추상화 누수, stringly-typed 코드, 불필요한 JSX 래핑 |
| **Agent 3: Efficiency**   | 효율성    | 불필요한 연산, 병렬화 누락, 핫패스 블로킹, N+1 패턴, TOCTOU 안티패턴, 메모리 누수          |

### Agent 1: Code Reuse Review

1. 새로 작성된 코드를 대체할 수 있는 **기존 유틸리티/헬퍼 검색** (utility 디렉토리, shared 모듈, 인접 파일)
2. 기존 기능을 중복하는 새 함수 플래그
3. 기존 유틸로 대체 가능한 인라인 로직 플래그 (수동 문자열 처리, 경로 핸들링, 환경 체크, 타입 가드 등)

### Agent 2: Code Quality Review

1. **중복 상태(Redundant state)**: 기존 상태를 복제하는 상태, 파생 가능한 캐시 값
2. **파라미터 스프롤(Parameter sprawl)**: 일반화 대신 새 파라미터를 계속 추가하는 패턴
3. **복붙 변형(Copy-paste with slight variation)**: 공유 추상화로 통합해야 할 유사 코드
4. **추상화 누수(Leaky abstractions)**: 내부 구현 노출, 기존 추상화 경계 위반
5. **Stringly-typed 코드**: 상수/enum/branded type 대신 raw string 사용
6. **불필요한 JSX 중첩**: 레이아웃에 기여하지 않는 래퍼 요소

### Agent 3: Efficiency Review

1. **불필요한 작업**: 중복 연산, 반복 파일 읽기, 중복 API 호출, N+1 패턴
2. **병렬화 누락**: 독립 작업의 순차 실행
3. **핫패스 블로트**: 시작/요청/렌더 핫패스에 블로킹 작업 추가
4. **무변경 업데이트**: 폴링/인터벌/이벤트 핸들러에서 변경 감지 없이 상태 업데이트
5. **불필요한 존재 확인**: TOCTOU 안티패턴 (존재 확인 후 조작 대신, 바로 조작하고 에러 처리)
6. **메모리**: 무제한 데이터 구조, 미정리, 이벤트 리스너 누수
7. **과도한 범위**: 일부만 필요한데 전체를 읽는 경우

## Phase 3: 이슈 수정

- 3개 에이전트 완료 대기 후 결과 종합
- 발견된 이슈를 **직접 코드 수정**
- 거짓 양성(false positive)은 건너뛰고 메모만 남김
- 최종 수정 요약 출력

## 특징

- **읽기 전용이 아님**: 리뷰 후 직접 수정까지 수행하는 쓰기 권한 스킬
- **병렬 실행**: 3개 에이전트가 동시 실행되어 순차보다 빠름
- **프론트엔드 특화 항목 포함**: JSX 중첩, 렌더 핫패스 등 React/웹 프론트엔드 체크 항목이 있어 웹 프로젝트를 주 타겟으로 설계됨
