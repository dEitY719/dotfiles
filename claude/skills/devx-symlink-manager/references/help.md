# Help — symlink-manager skill usage

## Usage

```
/symlink-manager <target_file>
```

## Arguments

| Argument | Description |
|---|---|
| `<target_file>` | Path to the config file to manage via symbolic link |
| `help` | Show this help message |

## Trigger Examples

- "xxx.yyy 파일을 symbolic link로 관리해"
- "dotfiles로 zzz 설정 파일 관리하고 싶어"
- "aaa.conf를 symbolic link로 설정해줘"
- "Manage [file] with dotfiles"
- "Set up symbolic link for [config]"
