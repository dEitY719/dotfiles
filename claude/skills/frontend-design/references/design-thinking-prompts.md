# Design Thinking Prompts — direction framing

Use these prompts before coding when direction is unclear.

## Purpose
- What user problem does this interface solve?
- Which moment matters most: discovery, conversion, comprehension, or control?

## Tone
Pick one strong direction and commit:
- brutally minimal
- maximalist chaos
- retro-futuristic
- organic / natural
- luxury / refined
- playful / toy-like
- editorial / magazine
- brutalist / raw
- art deco / geometric
- industrial / utilitarian

## Constraints
- Framework/runtime constraints
- Performance budget
- Accessibility requirements
- Device support (mobile-first, desktop-first, both)

## Differentiation
- What is the one memorable visual or interaction signature?
- Which default UI pattern will you intentionally avoid?

## Complexity Match Rule
- Minimal/refined concept: fewer elements, stricter spacing, high typographic precision.
- Maximalist/expressive concept: richer layering, stronger motion choreography, deliberate density.
