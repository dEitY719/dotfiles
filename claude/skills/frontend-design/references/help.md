# Help — frontend-design skill usage

/frontend-design — Build a distinctive frontend interface with intentional visual direction.

Usage:
  /frontend-design <request>
  /frontend-design help

Arguments:
  <request>  Description of the component/page/app/artifact to build.
  help       Show this message.

Examples:
  /frontend-design Build a conversion-focused fintech landing page for mobile + desktop.
  /frontend-design Create a React dashboard for incident response with a brutalist visual tone.

Workflow:
  1) Frame request and constraints
  2) Choose one clear aesthetic direction
  3) Implement working code
  4) Validate behavior, responsiveness, and accessibility basics
  5) Return result using the output format reference
