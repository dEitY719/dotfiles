# Brand Guidelines Skill

**Anthropic 공식 브랜딩 적용** - 프레젠테이션, 문서, 웹앱 등 모든 아티팩트에 통일된 회사 브랜드 아이덴티티를 적용합니다.

---

## 🎯 이 스킬이 뭔가요?

회사 프레젠테이션, 보고서, 웹앱, 포스터 등을 만들 때
Anthropic의 공식 브랜드 색상과 폰트를 자동으로 적용합니다.
무의미한 기본 스타일이 아닌 **전문적인 기업 이미지**를
보장합니다.

마치 디자인팀이 최종 검수를 한 것처럼, 모든 시각 자료가 일관된 Anthropic 브랜드 정체성을 따릅니다.

---

## 🔄 어떻게 작동하나요?

### 3단계 프로세스

```text
아티팩트 생성 → Brand Palette 적용 → 최종 검수 및 저장
```text

#### 1단계: 브랜드 색상 적용

- **주요 색상**:

  - Dark: `#141413` - 기본 텍스트, 어두운 배경

  - Light: `#faf9f5` - 밝은 배경

  - Mid Gray: `#b0aea5` - 보조 요소

  - Light Gray: `#e8e6dc` - 미세한 배경

- **악센트 색상**:

  - Orange: `#d97757` - 주요 강조

  - Blue: `#6a9bcc` - 보조 강조

  - Green: `#788c5d` - 3차 강조

#### 2단계: 폰트 적용

- **헤더**: Poppins (24pt 이상)

- **본문**: Lora (일반 텍스트)

- **폴백**: Arial(헤더), Georgia(본문) - 폰트 미설치 시

#### 3단계: 일관성 검증

모든 텍스트, 도형, 배경이 브랜드 가이드라인을 준수하는지 확인.

---

## 📊 Output (생산물)

### 브랜드 적용 아티팩트

Anthropic 브랜드 정체성이 적용된 최종 산출물:

```text
┌─────────────────────────────────┐
│   헤더 (Poppins, #141413)        │
│                                 │
│   본문 텍스트 (Lora, #b0aea5)    │
│                                 │
│   [강조 요소 - Orange #d97757]  │

│   [보조 요소 - Blue #6a9bcc]    │

└─────────────────────────────────┘
```

**지원 형식**:

- PowerPoint (PPTX)

- PDF 문서

- HTML/웹앱

- 모든 기타 아티팩트

---

## 💡 실제 사용 예시

### 요청

> "회사 분기별 성과 보고서 프레젠테이션을 만들어줘. 브랜딩도 적용해야 해."

### 결과

- **Title Slide**: Poppins 헤더 + Orange 강조선

- **Content Slides**: Lora 본문 + Green/Blue 데이터 시각화

- **Footer**: 일관된 Dark 배경, Light 텍스트

- **전체 톤**: Anthropic 공식 기업 이미지 유지

### 또 다른 예시

> "회사 웹사이트 헤더를 Anthropic 브랜드로 디자인해줘"

**결과**:

- 로고 주변: Mid Gray 배경

- 네비게이션 텍스트: Poppins, Dark

- 강조 버튼: Orange accent

- 전체: 기업 신뢰도와 전문성 표현

## 🚀 시작하기

### 사용자 입장 (Claude 사용할 때)

```text
"[아티팩트 설명]을 만들어줘. Anthropic 브랜드 가이드라인을 적용해야 해."
```text

또는

```text
"회사 프레젠테이션을 만들고 있어. 공식 브랜드 색상(주황색, 파란색, 초록색)과
Poppins/Lora 폰트로 적용해줄래?"
```

### 기대할 수 있는 것

1. **자동 색상 적용**: 모든 텍스트, 도형, 배경에 Anthropic 색상 사용
2. **일관된 폰트**: 제목은 Poppins, 본문은 Lora
3. **전문적 외관**: 사내 디자인팀이 승인한 것 같은 품질
4. **빠른 검수**: 브랜드 가이드 준수 여부 자동 검증

## 📚 스킬의 핵심 철학

> **"일관성이 신뢰를 만든다"**

- 모든 회사 커뮤니케이션은 **같은 시각 언어**를 사용해야 함

- 색상, 폰트, 간격 - 모든 것이 브랜드 정체성을 표현

- 사용자가 일일이 브랜드 가이드를 찾아보지 않아도 **자동 적용**

- 비용 효율: 디자인 수정 시간 대폭 단축

## ❓ FAQ

**Q: Poppins와 Lora 폰트가 설치되지 않은 경우는?**

A: 자동으로 Arial(헤더)과 Georgia(본문)로 대체됩니다. 시각적 품질 유지.

**Q: 특정 아티팩트에만 부분 적용할 수 있나요?**

A: 네! 제목만 Poppins, 특정 섹션만 Orange 강조 등 유연하게 적용 가능합니다.

**Q: 다른 색상도 사용해야 하면?**

A: Anthropic 브랜드 6색 이외의 색상이 필요한 경우, theme-factory 스킬로 커스텀 테마를 생성할 수 있습니다.

**Q: 모든 아티팩트 타입이 지원되나요?**

A: 네! PowerPoint, PDF, HTML, 웹앱, 포스터 등 거의 모든 시각 자료에 적용 가능합니다.
