# Slack GIF Creator

**Slack 최적화 GIF 생성** - 로딩 애니메이션, 이모지부터 메시지 GIF까지 멋진 애니메이션을 만듭니다.

---

## 🎯 이 스킬이 뭔가요?

Slack용 **애니메이션 GIF**를 만듭니다. 반복되는 멋진 로딩 바, 펄스 이모지, 깜빡이는 별 같은 시각적 효과를 Python으로 코딩합니다.

마치 애니메이션 아티스트처럼, 프레임을 하나하나 그려 생동감 있는 GIF를 완성합니다.

---

## 🔄 어떻게 작동하나요?

### 3단계 프로세스

```text
1. 요구사항 → 2. 애니메이션 설계 → 3. GIF 생성 및 최적화
```text

#### 1단계: 요구사항 명확히

질문:

- 이모지 GIF (128x128) 인가, 메시지 GIF (480x480) 인가?

- 어떤 애니메이션? (로딩, 펄스, 튕김, 스핀 등)

- 몇 초 길이?

- 색상은?

#### 2단계: 애니메이션 설계

**기본 프레임워크**:
```python
from PIL import Image, ImageDraw

# 프레임 생성
frames = []
for i in range(num_frames):
    frame = Image.new('RGB', (128, 128), background_color)
    draw = ImageDraw.Draw(frame)

    # 각 프레임마다 다른 위치/크기/색상으로 도형 그리기
    draw.ellipse([x, y, x+size, y+size], fill=color)

    frames.append(frame)
```

**지원 애니메이션**:

- 로딩 (회전 도형)

- 펄스 (커지고 작아짐)

- 튕김 (바운스)

- 회전 (스핀)

- 페이드 (나타나고 사라짐)

- 미끄럼 (슬라이드)

- 폭발 (파티클 효과)

#### 3단계: GIF 생성

**GIFBuilder 사용**:

```python
from core.gif_builder import GIFBuilder

builder = GIFBuilder(width=128, height=128, fps=10)
for frame in frames:
    builder.add_frame(frame)

builder.save('emoji.gif',
    num_colors=48,
    optimize_for_emoji=True)
```text

## 📊 Output (생산물)

### Slack 최적화 GIF

```text
┌─────────────┐
│  🔄 Frame 1 │  → 애니메이션 재생
│  🔄 Frame 2 │
│  🔄 Frame 3 │
│  ...        │
└─────────────┘

특징:

- 작은 파일 크기

- 매끄러운 애니메이션 (10-30 FPS)

- Slack에서 바로 작동

- 이모지 또는 메시지 GIF

```

**Slack 규격**:

- **이모지 GIF**: 128x128px, 3초 이하, 10-30 FPS

- **메시지 GIF**: 480x480px, 크기 제한 충분

## 💡 실제 사용 예시

### 예시 1: 로딩 애니메이션

#### 요청 (로딩)

> "Slack 이모지로 쓸 로딩 애니메이션을 만들어줄래?
> 파란색 도넛 모양으로."

#### 결과 (로딩)

```text
로딩 중... 🔄 (반복)

GIF 속성:

- 크기: 128x128

- 색상: 파란색 (#6a9bcc - Anthropic 브랜드)

- 애니메이션: 회전 도넛

- 길이: 2초

- FPS: 12

- 파일 크기: ~30KB

```

### 예시 2: 펄스 별 이모지

#### 요청 (별)

> "반짝이는 별 GIF를 만들어줄 수 있어?
> 노란색으로, 부드럽게 크기가 변했으면 좋겠어."

#### 결과 (펄스)

```text
⭐ (커짐) → ⭐ (작아짐) → 반복

GIF 속성:

- 크기: 128x128

- 색상: 노란색 (#FFD700)

- 애니메이션: 펄스 (heartbeat style)

- 길이: 1.5초

- FPS: 10 (부드러움)

- 파일 크기: ~20KB

```

### 예시 3: 복잡한 효과

#### 요청 (파티클)

> "메시지 GIF로 쓸 수 있는 파티클 폭발 애니메이션?"

#### 결과 (폭발)

```text
여러 입자가 중앙에서 방사형으로 폭발하고 페이드 아웃

GIF 속성:

- 크기: 480x480

- 색상: 다채로운 색상

- 애니메이션: 파티클 버스트

- 길이: 1초

- FPS: 20 (부드러움)

```

**최적화**:

## ❓ FAQ

**Q: 정말 복잡한 애니메이션도 가능한가요?**

A: 네! PIL로 거의 모든 것을 그릴 수 있습니다.

**Q: 파일 크기를 더 작게 할 수 있나요?**

A: 네! FPS 낮춤, 색상 감소, 길이 단축 등으로 최적화.

**Q: 이미지를 GIF에 포함할 수 있나요?**

A: 네! 업로드한 이미지를 로드해서 애니메이션 안에 포함 가능.

**Q: 어떤 색상이든 가능한가요?**

A: 네! RGB 값으로 임의의 색상 지정 가능.

**Q: 다른 크기도 가능한가요?**

A: 네! Slack 이모지(128x128) 또는 메시지(480x480) 외에도 가능.
