# Canvas Design Skill

**미니멀리즘 철학의 포스터부터 대담한 그래픽까지** - 박물관 수준의 시각 예술을 PDF/PNG로 생성합니다.

---

## 🎨 이 스킬이 뭔가요?

코드로 계산된 기하학적 형태가 아니라, **미학적 철학**을
먼저 정의한 후 그것을 시각으로 표현합니다. 각 색상,
형태, 배치는 의도적이며, 숨은 의미와 정교한
장인정신이 담겨 있습니다.

마치 미술관의 큐레이터가 갤러리를 설계하듯, 우리는 철학을 먼저 정하고 그것을 종이 위에 구현합니다.

---

## 🔄 어떻게 작동하나요?

### 2단계 프로세스

```text
미학 철학 정의 → 시각 표현 (PDF/PNG) 생성
```text

#### 1단계: Design Philosophy (미학 철학 정의)

- **이름 정하기**: 미술 운동처럼 이름 지음 (예: "Brutalist Joy", "Chromatic Silence")

- **4-6문단 철학 작성**:

  - 공간과 형태 (Space & Form)

  - 색상과 재질 (Color & Material)

  - 스케일과 리듬 (Scale & Rhythm)

  - 구성과 균형 (Composition & Balance)

  - 시각 계층 (Visual Hierarchy)

- **중요**: 정교한 장인정신 강조 - "수백 시간을 들인 작품", "마스터급 실행"

**예시**

```text
"Concrete Poetry" - 거대한 형태와 대담한 기하학으로 커뮤니케이션

"Chromatic Language" - 색상이 정보의 주요 언어

"Analog Meditation" - 조용한 여백, 사진과 감정

```

#### 2단계: Visual Expression (시각 표현)

철학을 이미지로 구현:

- 공간 구성: 대칭/비대칭, 그리드, 여백

- 색상 선택: 철학과 일맥상통

- 형태 배치: 의도적, 정밀함

- 최소한의 텍스트: "시각이 설명하고 글은 강조"

## 📊 Output (생산물)

### 1. Design Philosophy 마크다운

미학적 개념을 텍스트로 기술한 문서

### 2. Visual Artifact (PDF 또는 PNG)

```text
┌────────────────────────────────┐
│                                │
│  [박물관 수준의 시각 예술]       │
│                                │
│  - 완벽한 간격과 정렬          │

│  - 의도적인 색상 선택          │

│  - 은유적 의미 담김            │

│  - 최소한의 텍스트             │

│                                │
└────────────────────────────────┘
```text

**특징**:

- PDF 또는 PNG (선택 가능)

- 다중 페이지 옵션 (이야기 전개)

- 다운로드 가능, 인쇄 가능

## 💡 실제 사용 예시

### 요청

> "미니멀리즘 철학의 포스터를 만들어줘. 조용함과 공간감이 있어야 해."

### 결과

**Design Philosophy**:
```

"Serene Void" - 미니멀리즘의 철학

공간과 형태를 통해 침묵을 시각화합니다. 불필요한 요소를 제거하고
광활한 여백 속에 단 하나의 형태만 배치...

```text

**Visual Output**:

- 광활한 흰색 배경

- 중앙의 검은 원 (또는 간단한 기하학 형태)

- 균형 잡힌 여백

- 텍스트는 한 두 단어만 (선택적)

### 또 다른 예시

> "생태계의 순환을 표현하는 디자인"

**Design Philosophy**:
```

"Organic Systems" - 자연의 순환

둥근 형태, 유기적 배열, 색상은 자연에서... 정보는 공간과
배치로 전달되고, 텍스트는 거의 없습니다.

```text

**Visual Output**:

- 원형의 배열들

- 자연 색상 팔레트

- 계층적 구성

- 화살표나 라인으로 순환 표현

| **다중 페이지** | 이야기 같은 전개 가능 |
| **다운로드 가능** | PDF/PNG로 저장, 인쇄 또는 공유 |

## 🚀 시작하기

### 사용자 입장 (Claude 사용할 때)

```text
"[주제]를 표현하는 포스터를 만들어줘. 박물관 수준의 디자인으로."
```

또는

```text
"미니멀리즘 철학의 아트 작품을 만들어줄래? 의도적이고 정교해야 해."
```text

### 결과물 활용

1. **파일 다운로드**: PDF 또는 PNG로 저장
2. **인쇄**: 높은 해상도로 인쇄 가능
3. **공유**: 소셜 미디어, 프레젠테이션, 전시
4. **개선**: 마음에 들지 않으면 철학 수정 후 재생성

## 🛠️ 기술 스택

- **설계**: Canvas (PIL 또는 reportlab)

- **색상**: RGB 정밀 제어

- **형태**: 기하학적 형태 (원, 사각형, 선, 다각형)

- **폰트**: 선택한 폰트 (canvas-fonts 디렉토리)

- **출력**: PDF (reportlab) 또는 PNG (PIL)

## 📚 스킬의 핵심 철학

> **"미학은 의도에서 나온다"**

- 아무 디테일도 무작위가 아님 - 모두 계산됨

- 색상, 간격, 형태 - 모두 철학을 지지

- 미니멀하지만 정교 - 간단함이 아닌 엄선

- 박물관 기준: 이 작품이 갤러리에 전시되어도 손색없는 품질

## ❓ FAQ

**Q: 정말 박물관 수준의 디자인이 나오나요?**

A: 네! 철학을 정확히 정의하고, 마스터급 장인정신으로 실행하면 그 수준의 결과가 나옵니다.

**Q: 내 브랜드 색상을 사용할 수 있나요?**

A: 물론입니다! 철학에 맞는 색상을 선택하고, 그에 따라 비주얼을 생성합니다.

**Q: 다중 페이지 포스터는 가능한가요?**

A: 네! "커피 테이블 북" 같이 여러 페이지로 이야기를 전개할 수 있습니다.

**Q: 어떤 주제든 가능한가요?**

A: 네! 추상적 개념, 브랜드 아이덴티티, 자연, 문화, 역사 - 모든 것이 가능합니다.
