# DOCX Skill

**Word 문서 생성, 수정, 분석** - 계약서부터 제안서까지 완벽한 형식과 추적변경(tracked changes)을 지원합니다.

---

## 🎯 이 스킬이 뭔가요?

Word 문서(`.docx`) 파일을 만들고, 수정하고, 분석합니다.
간단한 편지부터 복잡한 계약서까지, 형식, 댓글,
추적변경을 모두 제어할 수 있습니다.
마치 전문 편집자처럼, Word의 고급 기능들(추적변경, 댓글, 스타일)을 활용합니다.

---

## 🔄 어떻게 작동하나요?

### 3가지 주요 워크플로우

```text
1. 새 문서 생성 → 2. 기존 문서 수정 → 3. 문서 분석
```text

#### 1. 새 문서 생성 (Creating New Documents)

**프로세스**:

- docx-js 라이브러리 사용

- Document, Paragraph, TextRun 컴포넌트로 구축

- 형식, 폰트, 스타일 적용

- `.docx`로 저장

**지원**:

- 기본 문단 및 제목

- 글릿, 번호 목록

- 표 (테이블)

- 이미지

- 스타일 및 포맷팅

#### 2. 기존 문서 수정 (Editing Existing Documents)

**추적변경 (Tracked Changes)** 사용:

- 원문 유지, 변경사항 추적

- 검수자가 수용/거부 가능

- 법적, 학술적 문서에 필수

**프로세스**:
1. Markdown으로 변경사항 계획
2. OOXML (XML) 직접 편집
3. 배치 단위로 변경 적용 (3-10개 변경씩)
4. 최종 검증

#### 3. 문서 분석 (Analyzing Content)

**가능한 작업**:

- 텍스트 추출 (Markdown으로)

- 댓글 읽기

- 스타일 분석

- 메타데이터 확인

**도구**: Pandoc (텍스트 추출), XML 직접 접근

## 📊 Output (생산물)

### 완성된 Word 문서

```text
┌────────────────────────────────┐
│  제목 (Bold, 14pt)             │
│                                │
│  본문 텍스트 (11pt, 표준)       │
│  - 글릿 항목 1                 │

│  - 글릿 항목 2                 │

│                                │
│  표:                           │
│  ┌─────┬─────┐                │
│  │ A   │ B   │                │
│  │ C   │ D   │                │
│  └─────┴─────┘                │
│                                │
│  [추적변경 또는 댓글 포함]      │
└────────────────────────────────┘
```

**특징**:

- 완벽한 형식

- 추적변경 지원

- 댓글 포함 가능

- 다운로드 가능

## 💡 실제 사용 예시

### 예시 1: 새 계약서 생성

#### 요청 (계약서)

> "NDA 템플릿 같은 계약서를 만들어줘. 서명 라인도 포함해서."

#### 결과 (NDA)

```text

- Word 문서 생성

- 표준 NDA 구조 (정의, 의무, 기간 등)

- 굵은 제목, 들여쓰기 섹션

- 서명 라인 (이름, 직책, 날짜)

- 완벽한 형식, 바로 인쇄 가능

```

### 예시 2: 제안서에 피드백 반영

#### 요청 (제안서)

> "이 제안서 2개를 병합하고, 섹션 3의 예산 부분을 업데이트해줄래? 추적변경으로."

#### 결과 (병합 후)

```text
원본 문서 유지, 변경사항:

- Section 1 (원본)

- Section 2 (원본)

- Section 3 (~~$50,000~~  → $75,000) [추적변경]

- Section 4 (원본)

검수자가 Word에서:
[✓ Accept] [✗ Reject] [△ Comment]
```

### 예시 3: 법적 검토

#### 요청 (검토)

> "이 계약서를 검토해줘. 문제 있는 부분에 댓글 달아줘."

#### 결과 (댓글 포함)

```text

- 텍스트 추출

- 문제 영역 식별

- Word 문서에 댓글 추가

- 수정 제안과 함께 반환

```

## 🛠️ 기술 스택

**문서 생성**: docx-js (JavaScript)

- Document, Paragraph, TextRun 컴포넌트

- 형식, 스타일 제어

- 간단하고 강력함

**문서 수정**: OOXML (Python)

- XML 직접 조작

- 추적변경 구현

- 정밀한 제어

**분석**: Pandoc

- 텍스트 추출

- 형식 보존

- 마크다운 변환

## ❓ FAQ

**Q: 정말 추적변경으로 정확하게 수정되나요?**

A: 네! XML 수준에서 정밀하게 제어하므로 완벽합니다.

**Q: 기존 Word 형식을 유지할 수 있나요?**

A: 네! 기존 문서를 로드하면 형식을 보존한 채로 수정합니다.

**Q: 댓글도 추가할 수 있나요?**

A: 네, 특정 텍스트에 댓글을 추가할 수 있습니다.

**Q: 다국어 문서도 지원되나요?**

A: 네! Word가 지원하는 모든 언어와 문자를 사용할 수 있습니다.

**Q: PDF로 변환도 가능한가요?**

A: 네! Word → PDF 변환이 가능합니다. (LibreOffice 사용)
