# PPTX Skill

**PowerPoint 생성, 수정, 분석** - 비즈니스 프레젠테이션부터 교육 자료까지 완벽한 슬라이드를 만듭니다.

---

## 🎯 이 스킬이 뭔가요?

PowerPoint 프레젠테이션(`.pptx`) 파일을 만들고, 수정하고,
분석합니다. 단순한 텍스트 슬라이드부터 차트, 이미지,
애니메이션이 들어간 복잡한 프레젠테이션까지 모두
가능합니다.

마치 전문 디자이너와 함께 작업하는 것처럼, **색상, 폰트, 레이아웃**을 완벽하게 제어합니다.

---

## 🔄 어떻게 작동하나요?

### 3가지 주요 워크플로우

```text
1. 새 프레젠테이션 생성 → 2. 기존 프레젠테이션 수정 → 3. 프레젠테이션 분석
```text

#### 1. 새 프레젠테이션 생성 (Creating Presentations)

**프로세스**:
1. HTML 슬라이드 작성 (각 슬라이드당 1개)
2. PptxGenJS로 PowerPoint 변환
3. 차트, 이미지, 형식 추가
4. 저장 및 검증

**특징**:

- HTML 기반 설계 (CSS 지원)

- 자동 레이아웃

- 차트, 표, 이미지 삽입

- 애니메이션 지원

#### 2. 기존 프레젠테이션 수정 (Editing Presentations)

**프로세스**:
1. PPTX를 XML로 언팩
2. 슬라이드 텍스트/형식 수정
3. 슬라이드 재배열
4. 이미지/차트 업데이트
5. 다시 팩하기

**지원**:

- 텍스트 편집

- 슬라이드 재배열

- 스타일 변경

- 이미지 교체

#### 3. 프레젠테이션 분석 (Analyzing Presentations)

**기능**:

- 텍스트 추출 (Markdown)

- 슬라이드 개수 확인

- 썸네일 생성

- 레이아웃 분석

---

## 📊 Output (생산물)

### 완성된 PowerPoint

```text
┌────────────────────────────────┐
│  Slide 1: Title Slide          │
│  [큰 제목, 부제]                │
│                                │
│  Slide 2: Content              │
│  [제목, 글릿 항목들]            │
│                                │
│  Slide 3: Data Visualization   │
│  [차트, 그래프, 표]             │
│                                │
│  [완벽한 형식, 색상, 폰트]      │
│  [다운로드 가능, 인쇄 가능]      │
└────────────────────────────────┘
```

**특징**:

- 완벽한 형식

- 일관된 디자인

- 모든 기능 (차트, 이미지, 애니메이션)

- 모든 플랫폼 호환

## 💡 실제 사용 예시

### 예시 1: 분기 비즈니스 프레젠테이션

#### 요청 (사업)

> "Q1 사업 보고 프레젠테이션을 만들어줄 수 있어?
> 매출, ROI, 팀 성과가 포함되어야 해."

#### 결과 (사업)

```text
Slide 1: 제목 (회사 로고, Q1 2025)
Slide 2: 목차
Slide 3: 매출 분석 (차트 포함)
Slide 4: ROI 비교 (표 포함)
Slide 5: 팀 성과 (아이콘 + 텍스트)
Slide 6: 결론 및 Q&A

색상: 브랜드 컬러 적용
폰트: 전문적 조합
레이아웃: 일관된 디자인
```

### 예시 2: 교육용 프레젠테이션

#### 요청 (교육)

> "Python 기초 강의 슬라이드를 만들어줄래?
> 15개 슬라이드, 코드 예시 포함."

#### 결과 (교육용)

```text
Slide 1: 제목 (강의명, 강사명)
Slide 2: 학습 목표
Slides 3-14: 각 주제별 내용
  - 개념 설명

  - 코드 예시 (모노스페이스 폰트)

  - 연습 문제

Slide 15: 요약 및 다음 단계

스타일: 교육용 (명확한 계층, 읽기 쉬운 폰트)
```

### 예시 3: 템플릿 기반 프레젠테이션

#### 요청 (템플)

> "회사 템플릿을 사용해서 새로운 제안서를 만들어줄 수 있어?"

#### 프로세스 (템플릿)

1. 템플릿 분석 (슬라이드 레이아웃 이해)
2. 필요한 슬라이드 선택/복사
3. 컨텐츠 교체
4. 이미지/차트 업데이트

#### 결과 (템플)

```text
템플릿의 디자인과 스타일 유지하며
새로운 내용으로 프레젠테이션 완성
```

```text

```

또는 템플릿 기반:

```text
"이 템플릿을 사용해서 [새로운 주제]의 프레젠테이션 만들어줄래?"
```text

### 기대할 수 있는 것

1. **완벽한 형식**: 전문성 보장
2. **일관된 디자인**: 모든 슬라이드 조화
3. **고급 기능**: 차트, 이미지, 애니메이션
4. **빠른 수정**: 요청하면 즉시 반영
5. **다운로드**: 바로 사용 가능

## 🛠️ 기술 스택

**새 프레젠테이션**: PptxGenJS (JavaScript)

- HTML 기반 슬라이드 작성

- 자동 레이아웃

- 차트/표 생성

**편집/분석**: OOXML (Python)

- XML 직접 조작

- 정밀한 제어

- 슬라이드 재배열

**변환**: LibreOffice + Poppler

- PPTX → PDF

- PPTX → 이미지

## ❓ FAQ

**Q: 정말 프로페셔널한 디자인이 나오나요?**

A: 네! 색상 선택, 폰트 조합, 레이아웃 모두 최적화됩니다.

**Q: 회사 템플릿을 사용할 수 있나요?**

A: 네! 기존 템플릿을 분석한 후 그 스타일로 작성합니다.

**Q: 차트는 자동으로 생성되나요?**

A: 네! 데이터만 주면 차트 자동 생성 및 레이아웃.

**Q: 애니메이션도 가능한가요?**

A: 네! 전환, 페이드, 확대 등 다양한 애니메이션.

**Q: 수정은 몇 번까지?**

A: 무제한! "슬라이드 3 크기 조정해줄래?" 하면 즉시 반영.
