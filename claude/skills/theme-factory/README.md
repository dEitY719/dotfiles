# Theme Factory

**프레젠테이션/아티팩트 테마 적용** - 10개의 전문적인 테마 중 선택하거나 커스텀 테마를 만들어 아티팩트를 꾸밉니다.

---

## 🎯 이 스킬이 뭔가요?

슬라이드, 문서, 웹앱 등 모든 아티팩트에 **일관된
스타일**을 적용합니다. 10개의 기획된 테마 중 선택하거나,
자신의 색상과 폰트로 커스텀 테마를 만듭니다.

마치 인테리어 디자이너가 색상 팔레트를 제안하듯, 전문적인 색상과 폰트 조합을 제시합니다.

---

## 🔄 어떻게 작동하나요?

### 2단계 프로세스

```text
1. 테마 선택 (또는 커스텀 생성) → 2. 아티팩트에 적용
```text

#### 1단계: 테마 선택 또는 생성

**기존 테마 (10개)**:
1. **Ocean Depths** - 전문성, 바다색 (파란색 톤)

2. **Sunset Boulevard** - 따뜻함, 석양색 (주황/빨강)

3. **Forest Canopy** - 자연스러움, 숲색 (녹색 톤)

4. **Modern Minimalist** - 단순함, 회색조

5. **Golden Hour** - 우아함, 황금색 톤

6. **Arctic Frost** - 차가움, 얼음색 (파란/흰색)

7. **Desert Rose** - 부드러움, 먼지 핑크

8. **Tech Innovation** - 대담함, 기술색 (검정/밝은색)

9. **Botanical Garden** - 신선함, 정원색 (초록/연분홍)

10. **Midnight Galaxy** - 드라마틱, 심우주색 (검정/보라)

**또는 커스텀 테마**:

- 색상 팔레트 설정

- 폰트 쌍 선택

- 이름 정의

- 검토 및 적용

#### 2단계: 아티팩트에 적용

선택한 테마의:

- **색상**: 헤더, 배경, 악센트, 텍스트

- **폰트**: 제목, 본문, 강조

모든 섹션에 일관되게 적용.

## 📊 Output (생산물)

### 테마 적용 완료 아티팩트

```text
┌────────────────────────────────┐
│  [테마 색상 적용]               │
│  헤더: 주색상                  │
│  본문: 본문 폰트               │
│  강조: 악센트 색상             │
│                                │
│  전체: 일관된 디자인            │
│  색상 조화, 가독성 보장         │
└────────────────────────────────┘
```

**특징**:

- 색상 일관성

- 전문적 외관

- 가독성 보증

- 즉시 적용 가능

## 💡 실제 사용 예시

### 예시 1: 프레젠테이션 테마 선택

#### 요청 (선택)

> "만든 분기 보고서 프레젠테이션에 테마를 입혀줄 수 있어?
> 전문적이고 차분했으면 좋겠어."

#### 프로세스 (선택)

1. theme-showcase.pdf 보기
2. "Ocean Depths 맘에 드는데 이걸로"
3. 프레젠테이션 모든 슬라이드에 Ocean Depths 색상/폰트 적용

#### 결과

```text
모든 슬라이드:

- 헤더: 깊은 파란색 (Ocean Depths 주색상)

- 본문: 우아한 폰트

- 차트: 파란색 톤의 그래디언트

- 강조: 밝은 파란색 액센트

```

### 예시 2: 커스텀 테마 생성

#### 요청 (브랜드)

> "우리 브랜드 컬러(보라색, 금색)로 커스텀 테마를 만들어줄 수 있어?"

#### 프로세스 (커스텀)

1. Claude가 커스텀 테마 제안

```text
   이름: "Royal Elegance"
   주색상: #7c3aed (보라색)
   악센트: #fbbf24 (금색)
   폰트: Poppins (제목), Lora (본문)
```

1. 검토 및 승인
2. 모든 아티팩트에 적용

#### 결과 (커스텀)

```text
new-brand-theme.json
모든 문서, 슬라이드, 웹앱에
"Royal Elegance" 테마 적용 완료
```

### 예시 3: 웹앱에 테마 적용

#### 요청 (웹)

> "대시보드 웹앱이 완성됐어. Botanical Garden 테마로 꾸며줄 수 있어?"

#### 결과 (웹앱)

```text
CSS 변수에 Botanical Garden 색상 적용:

- 배경: 크림색

- 제목: 진한 녹색

- 강조: 연분홍

- 버튼: 녹색 톤

```

## 🛠️ 기술 스택

**테마 정의**:

- JSON 기반 색상 팔레트

- 폰트 스택 (시스템 폰트)

- 명명 규칙 (색상, 크기 등)

**적용**:

- CSS 변수 (웹앱)

- PPTX 테마 (PowerPoint)

- 스타일 설정 (Word, 문서)

**관리**:

- themes/ 디렉토리

- theme-showcase.pdf

- theme.json 파일

## 📚 스킬의 핵심 철학

> **"색상이 브랜드를 만든다"**

- 일관된 색상 팔레트 = 전문성

- 색상 대비 = 가독성

- 폰트 쌍 = 성격

- 선택의 의도성이 품질을 결정

## ❓ FAQ

**Q: 10개 테마가 아니어도 되나요?**

A: 커스텀 테마를 만들 수 있습니다! 고유한 색상과 폰트로 새 테마 생성.

**Q: 기존 테마를 수정할 수 있나요?**

A: 네! "Ocean Depths를 기반으로 주색상을 더 밝게 해줄 수 있어?" 하면 수정.

**Q: 여러 아티팩트에 같은 테마?**

A: 네! "모든 프레젠테이션과 문서에 Sunset Boulevard 적용"하면 일괄 적용.

**Q: 색상을 정확히 지정할 수 있나요?**

A: 네! RGB 또는 HEX 값으로 정확한 색상 지정 가능.

**Q: 테마를 저장할 수 있나요?**

A: 네! theme.json으로 저장되어 재사용 가능.
